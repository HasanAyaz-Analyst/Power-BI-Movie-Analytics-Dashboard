# Power-BI-Movie-Analytics-Dashboard
Overview

This project presents an interactive Power BI Movie Analytics Dashboard designed to analyze trends and performance metrics within the film industry. The dashboard combines financial, popularity, and geographic insights using a clean IBM Carbon–inspired dark theme for a professional and modern look.

The goal of this project is to demonstrate data modeling, DAX calculations, and effective data storytelling using Power BI.

Dashboard Insights

The dashboard provides insights into:

Total number of movies

Budget, gross profit, and return on investment (ROI)

Facebook likes by directors and actors

Movie count and profit distribution by country

Budget trends across years

Key Features

Interactive slicers for Year, Genre, Country, and Movie Title

KPI cards for high-level metrics

Bar charts for actor and director popularity

Map visualization for geographic analysis

IBM Carbon–inspired dark theme using a custom JSON file

Clean layout optimized for storytelling and decision-making
